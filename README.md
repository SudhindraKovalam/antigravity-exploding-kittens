# antigravity-exploding-kittens

This repository hosts the development of an **experimental web version of the Exploding Kittens card game**.

## Project Goal

The purpose of this project is two-fold:

1. **Web Adaptation**: Build a fully functional, interactive web application for playing Exploding Kittens.
2. **Antigravity Experiment**: Serve as a testbed for evaluating and demonstrating **antigravity-based development** workflows and capabilities.
